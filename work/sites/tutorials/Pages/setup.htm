Tutorials Setup

Program Installs

- If you don't have the full version of eyeon Digital Fusion v4.03b installed on your computer, you can install the fully functional demo so that you are able to complete all of the tutorials on this disc.

Setup file location:

\Setup\Eyeon\digitalfusion403b_demo.exe

 

- The Smokey tutorial requires the SpeedSix Monsters v4.5.4 plug-ins for Digital Fusion. If you need to install the demo version of these plug-ins, please read the readme file and then install the demo.

Setup file location:

\Setup\SpeedSix\MonstersDFV4.5.4.exe

 

Tutorial Materials

- There is a folder called M3D_Materials on this disc. It contains all of the files used to complete the tutorials. Please copy this folder to your hard drive (ex: C:\M3D_Materials).

- If you are new to Digital Fusion please check out out Common Procedures section.
